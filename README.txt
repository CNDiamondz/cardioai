CardioAI – Standalone App
==========================

SETUP
-----
1. Unzip this file
2. Open index.html in any modern browser (Chrome, Firefox, Edge, Safari)
3. No server, no build step, no installation required

USAGE
-----
- Upload an ECG image (PNG/JPG), PDF, or text/CSV file
- Add patient context (age, sex, symptoms) for personalized results
- Use Quick Questions for common follow-ups
- Use the QTc Calculator tab
- Export any analysis as a text report

REQUIREMENTS
------------
- Internet connection (calls the AI API)
- Modern browser with JavaScript enabled
